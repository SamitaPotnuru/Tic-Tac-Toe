# 🎮 Tic Tac Toe

A standalone single-file Tic Tac Toe game built with HTML, CSS, and JavaScript.

## Features

- **Two Player Mode** — Play X vs O locally
- **Score Tracking** — Persistent scores saved in localStorage
- **Win Detection** — Highlights winning cells with animation
- **Draw Detection** — Detects when the board is full
- **Restart** — Reset the board without losing scores

## How to Play

1. Open `standalone.html` in any browser
2. Player X goes first — click any empty cell
3. Players alternate turns until someone wins or it's a draw
4. Click **Restart** to play again

## Tech

- Pure HTML/CSS/JS — no dependencies
- CSS Grid layout with hover/win animations
- localStorage for score persistence
